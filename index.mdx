title: "Trustee — Attestation and Secret Delivery"
description: "Trusted components for remote attestation and secret delivery to confidential guests in Trusted Execution Environments (TEEs)."
---

Trustee is a suite of components that attests confidential guests running in hardware TEEs and securely delivers secrets to them. It implements the [RATS architecture](https://www.ietf.org/archive/id/draft-ietf-rats-architecture-22.html) and supports Intel TDX, Intel SGX, AMD SEV-SNP, ARM CCA, IBM Secure Execution, NVIDIA GPU attestation, TPM, and more.

## How it works

<Steps>
  <Step title="Guest requests attestation">
    A confidential guest running in a TEE contacts the KBS to request a secret resource. The KBS issues a challenge nonce.
  </Step>
  <Step title="TEE generates evidence">
    The guest generates hardware-signed attestation evidence binding the nonce and an ephemeral public key to the TEE measurement.
  </Step>
  <Step title="Evidence is verified">
    The Attestation Service verifies the TEE evidence against reference values from the RVPS and evaluates it with OPA policies.
  </Step>
  <Step title="Secret is delivered">
    After successful attestation, the KBS encrypts the requested secret with the guest's TEE public key and delivers it securely.
  </Step>
</Steps>

## Supported TEE platforms

Trustee supports a wide range of hardware TEE platforms:

| Platform | Description |
|---|---|
| Intel TDX | Intel Trust Domain Extensions |
| Intel SGX | Intel Software Guard Extensions |
| AMD SEV-SNP | AMD Secure Encrypted Virtualization — Secure Nested Paging |
| ARM CCA | Confidential Compute Architecture |
| IBM SE | IBM Z Secure Execution |
| NVIDIA | NVIDIA GPU attestation |
| Azure SNP vTPM | Azure AMD SEV-SNP with vTPM |
| Azure TDX vTPM | Azure Intel TDX with vTPM |
| TPM | Trusted Platform Module |

<Note>
Trustee was developed for the [Confidential Containers](https://github.com/confidential-containers) project but can be used with any confidential computing workload or hardware platform.
</Note>
